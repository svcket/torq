# North Star Profile — TORQ

## 1. Project Identity
**TORQ**: The Cinematic Engine for Movement Culture.

## 2. Core Purpose
TORQ transforms fragmented subcultures (drift, bike, skate, bmx) into a unified, high-fidelity digital archive and event ecosystem. It brings the precision of professional editorial storytelling to the grit of street-level movement.

## 3. The "Soul"
**Controlled Kinesis**. The platform should feel alive and moving, but always with the restraint of high-fidelity photography and editorial discipline. It is not a social network; it is a cultural monument.

## 4. North Star Metric (NSM)
**Momentum Depth**: Measured by the engagement with editorial long-forms and the conversion to event discovery/merchandise loops.

## 5. The Quality Bar
- **The "No-Startup" Rule**: Avoid playful rounded corners, bright purples, or chirpy UI copy.
- **Cinematic Primacy**: Every screen should look like a frame from a high-budget car film.
- **8pt Grid Discipline**: Absolute alignment. No "near-enough" padding.
- **Typographic Authority**: Anton used for impact; Inter used for intelligence.

## 6. Input Metrics (Breadth / Depth / Quality)
- **Archive Growth**: Frequency and quality of "Past Scenes" contributions.
- **Event Velocity**: Speed from event announcement to fully hydrated discovery.
- **Merch Resonance**: Narrative-driven conversion versus generic shop behavior.

## 7. Operational Health
- **Uptime Baseline**: 99.9% availability for event discovery.
- **Asset Integrity**: Zero broken media links or low-resolution placeholders.
