# Project Context — TORQ

Distilled from user directives as of April 16, 2026. This data serves as the persistent "Memory" for the Project Agent during Phase 1.

## 1. Domain Coverage
TORQ is a collective of sub-cultures centered on **Movement**:
- **Drift Culture**: Night meets, professional and amateur drifting.
- **Car Meets**: Enthusiast gatherings and showcase logic.
- **Superbikes & Biking**: Performance and stunt culture.
- **Skating / BMX**: Urban movement and street-level scenes.
- **Water/Lifestyle**: Selected high-octane water sports and associated social tiers.

## 2. Core Modules
- **Event Discovery**: Real-time and upcoming movement gatherings.
- **Archives**: High-fidelity records of "Past Scenes."
- **Editorial**: Long-form storytelling and cultural deep-dives.
- **Merchandise**: Community-led apparel and capsule drops.
- **Partner Ecosystem**: Sponsors, builders, and cultural stakeholders.

## 3. Brand & Aesthetic Constants
- **Typography**: Anton (Display) / Inter (Sans).
- **Primary Color**: #EF4826 (Torq Red).
- **Core Principle**: "Premium, flat, sharp, editorial styling."
- **Vibe Constraint**: "Cinematic but controlled; not playful startup energy."

## 4. Setup Assumptions
- The project will use Next.js for high-fidelity rendering and Tailwind CSS v4 for utility-first styling.
- Motion will be handled by Framer Motion and Lenis for smooth vertical transitions.
- The 8pt Grid Protocol must be respected across all layouts.
